Поиск и оплата штрафов ГИБДД. Партнерская программа

== Description ==
Плагин позволяет искать и оплачивать штрафы ГИБДД по номеру автомобиля, номеру водительского удостоверения и номеру штрафа (УИН). Установивший плагин будет зарабатывать 3% с каждого оплаченного штрафа.

== Installation ==
Установите плагин на сайт сайт. Активируйте его. Перейдите по ссылке https://www.driver-helper.ru/shtrafy-gibdd/widget и получите "Код API для виджета Word" добавьте его в соответствующую форму в настройках плагина
